# Isabela Nunes Nutri - Modelo de Landing Page

Este repositório apresenta um modelo de landing page desenvolvido para nutricionistas, ideal para profissionais que desejam uma presença online elegante e funcional. Este projeto serve como um exemplo de portfólio da **Hyperpage**, uma agência especializada na criação de sites, demonstrando a qualidade e a capacidade de entrega da nossa agência.

## Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as tecnologias essenciais da web para garantir leveza e compatibilidade:

*   **HTML:** Estruturação do conteúdo da página.
*   **CSS:** Estilização e design responsivo, garantindo uma ótima visualização em diferentes dispositivos.

## Visualizar Projeto

Você pode visualizar este modelo de landing page diretamente pelo GitHub Pages aqui: [https://liracoder.github.io/isabelanunes-nutri/](https://liracoder.github.io/isabelanunes-nutri/)

Para visualizar localmente, basta abrir o arquivo `index.html` em qualquer navegador web.
